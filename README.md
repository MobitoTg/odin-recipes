# odin-recipes

Odin Recipes is a small collection of recipe pages built while learning HTML. It includes three finished recipe pages and a reusable template to keep new recipes consistent.

What I built
- Three favorite recipes: `newyorkstrip.html`, `shortrib.html`, and one additional recipe page.
- A `template.html` file used to scaffold new recipe pages so they share structure, comments, and link behavior.

Workflow note
- I created the first recipe manually, then extracted the common structure into `recipes/template.html`. From that template I duplicated and filled new files to ensure uniform markup and UX across the site.

See `https://github.com/MobitoTg/odin-recipes/recipes/readme.md` for more details about the `recipes/` folder.
