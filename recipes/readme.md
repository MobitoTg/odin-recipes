# Recipes

This directory contains individual recipe pages and a blank recipe template used by the site.

Files:
- `newyorkstrip.html` — Finished recipe page for New York Strip steak (includes image, attribution, ingredients, and steps).
- `shortrib.html` — Recipe page scaffold for short ribs (editable recipe file).
- `template.html` — Minimal HTML template to copy when adding a new recipe.

How to add a new recipe:
1. Duplicate `template.html` and rename to your recipe filename (e.g., `chicken-parmesan.html`).
2. Update the `<title>`, `<h1>`, image `src`/`alt`, attribution link, ingredients list, and steps.
3. Add a link to the recipe from `../index.html` if desired.

Notes:
- External links that open in a new tab use `target="_blank"`.
- Keep markup simple and consistent with existing recipe pages.

Happy cooking!
